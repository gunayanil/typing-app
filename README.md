# Full Stack Serveless Typing Game

- React
- Styled-Components
- Serverless Functions (Netlify)
- Auth0
- Airtable

 Wrote serverless functions for all logic related to score. Saved scores in an Airtable table. Implemented authentication and authorization with Auth0.
 Deployed on Netlify.
